## 一、章节结构

### 第一章 绪论

建议内容：
- 具身智能与桌面级机器人系统背景
- 感知-决策-执行一体化原型系统的研究意义

- 本文研究目标、技术路线和主要贡献

### 第二章 相关工作

建议内容：
- LeRobot 生态与机器人数据格式
- LangGraph 多 Agent 编排方法
- 视觉语言模型在机器人感知中的应用
- SmolVLA 轻量化 VLA 模型相关工作

### 第三章 系统架构设计

建议内容：
- 系统总体架构

### 第四章 决策层 Agent 设计

建议内容：
- LangGraph 状态结构设计
- `task_planner`、`scene_analyzer`、`action_decider`、`verifier` 的职责

- 模型可切换机制

### 第五章 系统实现与验证

建议内容：
- 硬件平台与软件环境

- 测试与实验结果分析

### 第六章 总结与展望

建议内容：
- 当前平台化设计价值总结
- 模块化扩展能力总结
